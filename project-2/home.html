Rotoscoping Frame by Frame: One Method

Here is a short frame by frame animation I made for Introduction to Animation. If you study hard and only give up when the lack of sleep is causing hallucinations you might be able to put something together like this at the ed of this Instructable.

I've put together this table so that you can see some of the divisions of work, money and happiness which come along with a project like this.

Pre-Production Production Post-Production

Work

When you are making something that is hand drawn and then painted and then scanned and compiled into a moving picture there are a lot of repetitious tasks which must be performed in order. This is the amount of time you should set aside to complete each phase of the project.

1-2 Hours 16-24 Hours 4-5 Hours

Money

I'm not saying this is the cheapest way to get this done. I use a lot of madern technology like LED lightboards and an editing suite. I am sure you could find ways to do this project without spending a dime but this is not that Instructable.
